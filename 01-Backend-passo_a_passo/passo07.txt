
Vamos implementar segurança com o "bcryptjs" 
Instalação na raiz
npm install bcryptjs
npm install -D @types/bcryptjs

Ajustar o Controller "UserController.ts", vamos importar o hash e colocar antes o hash antes de salvar ele no banco
Depois vamos testar pelo Postman, depois para ver se no banco foi salvo com hash
comando (docker exec -it med_schedule_db mysql -u root -p -e "USE med_schedule; SELECT * FROM users;)


Agora vamos colocar o JWT para ter o token, vamos instalar o JWT e vamos criar um aruivo Auth no controller.
Instalação na raiz
npm install jsonwebtoken
npm install -D @types/jsonwebtoken

Criação do Auth "src/controllers/AuthController.ts"
