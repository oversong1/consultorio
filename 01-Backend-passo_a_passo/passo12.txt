

Bom foi feito o create do Doctor, e foi testado e funcionou, no entando para facilitar o processo, não vamos fazer separado vamos deixar a logica e a execução juntos. 
Sendo assim um service e um controller para o Doctor.


E para isso vamos criar um arquivo novo chamado "src/services/DoctorService.ts", 
aqui vem o ponto vamos aproveitar o inicio de cadastro do outro arquivo o "CreateDoctorService.ts".
E vamos adicionar as outras logicas no novo arquivo.

E tambem vamos criar o arquivo "src/controllers/DoctorController.ts", 
aqui vem o ponto vamos aproveitar a logica do "CreateDoctorController.ts".
E vamos adicionar as outras logicas no novo arquivo.

E por fim vamos ajustar a "route.ts" com as rotas dos medicos.

OBS: Agora que ajustamos a logica vamos deletar os aqruivos "DoctorService.ts e DoctorController.ts"

Por fim vamos testar se esta tudo funcionando no "PostMan"

Cadastro Metodo "POST" http://localhost:3333/doctors/
Body->raw->json 
Corpo da requisição:
"
{
  "name": "Dr. Marcos",
  "crm": "789123-SP",
  "email": "marcos@hospital.com",
  "specialty": "Clinico Geral"
}
"

Ver Medicos Geral ou individual com ID
Metodo "GET" : http://localhost:3333/doctors/
Metodo "GET" : http://localhost:3333/doctors/013694af-7db6-41f5-9f17-59d5c9e0ac57

Atualizar Medico
Metodo "PUT" : http://localhost:3333/doctors/0ece1b1f-94c1-4ad5-b188-6ef5b458c56d
Body->raw->json 
Corpo da requisição colocar o que vamos atualizar exemplo:
"
{
  "name": "Dr. Marcos  Fonseca",
  "specialty": "Diagnóstico"
}
"

Deletar Medico com o ID
Metodo "DELETE" : http://localhost:3333/doctors/0ece1b1f-94c1-4ad5-b188-6ef5b458c56d
